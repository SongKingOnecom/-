# 军事制度史两部

由 DeepSeek-V4-Pro（第一作者）在 Zhao（通讯作者）的引导下完成的军事制度史研究系列，旨在填补中文世界在世界军事制度通史与中国历代精锐部队制度史两个方向上的空白。

书籍内容没有经过严格的校准，细节都有偏颇；并非严肃的学术作品，不做商业用途，唯一目的是供大家讨论，思考；后续将根据反馈的意见重新修订。

---

## 《世界军事制度与军事文化通史——从城邦到总动员》

**约公元前 3000 年—公元 1914 年 | 四卷 · 六十二编 · 二百四十一章 · 约一百五十万字**

首部以全球视野系统考察军事制度演化的中文通史著作。以"制度如何将分散的个体组织为协调的暴力"为核心问题，跨越五千年、覆盖欧亚非美四大洲，从苏美尔城邦写到一战前夕。

- **第一卷 · 古代**（约前 3000 年—476 年）：两河流域、古埃及、中国先秦至魏晋南北朝、古希腊罗马、波斯、印度、游牧与古代文明
- **第二卷 · 中世**（476—约 1500 年）：拜占庭军区制、西欧封建、隋唐府兵制、宋辽金夏、蒙古帝国、日本朝鲜、非洲与美洲
- **第三卷 · 近世**（约 1500—1789 年）：欧洲军事革命、明朝卫所制、清朝八旗制、日本近世、奥斯曼、莫卧儿、萨法维
- **第四卷 · 近代**（1789—1914 年）：法国大革命全民皆兵、拿破仑战争、19 世纪各国制度演进、中国晚清转型、一战前夕

**方法论**：以制度分析而非战史叙事为核心，通过跨文明比较揭示征兵、编制、指挥、后勤等制度要素的普适性因果机制。

## 《中国历代精锐部队考论》

**从夏商到清末 | 五编 · 三十四章 · 三附录**

系统考察中国历史上所有有文献可考的精锐部队——从商代"王作三师"到清末新军，约一百九十余支。每支部队详考其创立背景、编制结构、兵源选拔、训练方式、主要战绩与制度结局。

- **第一编 · 先秦**：夏商早期精锐雏形、西周王师、春秋诸侯私卒、战国魏武卒与秦锐士
- **第二编 · 秦汉魏晋**：西汉南北军、东汉五校、三国虎豹骑与无当飞军、北府兵
- **第三编 · 隋唐五代**：隋骁果、唐玄甲军与神策军、藩镇牙兵
- **第四编 · 宋辽金元**：北宋禁军与西军、辽皮室军、金铁浮屠、蒙古怯薛
- **第五编 · 明清近代**：明三大营、清八旗与湘军、清末新军

**附录**：中国历代精锐部队一览表、主要引用书目与参考文献、相关出土简帛与文物。
